1 安装Hugo

Hugo官方文档：https://gohugo.io/getting-started/installing/

Hugo中文文档：https://www.gohugo.org/doc/overview/introduction/

1.1 二进制方式安装

下载二进制文件放置到指定目录下，然后将该文件路径添加至环境变量

1.2 源码方式安装

依赖工具：

  1. Git
  2. Go (at least Go 1.11)

下载Hugo源码并编译

$ git clone https://github.com/gohugoio/hugo.git

$ cd hugo/

$ go install

解决go包管理代理网址无法访问：proxy.golang.org

默认使用的是proxy.golang.org，在国内无法访问

$ go install

go: github.com/BurntSushi/locker@v0.0.0-20171006230638-a6e239ea1c69: Get "https://proxy.golang.org/github.com/%21burnt%21sushi/locker/@v/v0.0.0-20171006230638-a6e239ea1c69.mod": dial tcp 34.64.4.81:443: connectex: A connection attempt failed because the connected party did not properly respond after a period of time, or established connection failed because connected host has failed to respond.

解决方法： 换一个国内能访问的代理地址：https://goproxy.cn

go env -w GOPROXY=https://goproxy.cn

1.3 验证是否安装成功

$ hugo version

Hugo Static Site Generator v0.75.0-DEV windows/amd64 BuildDate: unknown

2 创建博客站点

$ hugo new site ycblog

Congratulations! Your new Hugo site is created in D:\ycblog.

$ cd ycblog/

站点目录结构

config.toml
content/
data/
layouts/
static/
themes/

3 新建文章

$ hugo new post/buildhugoblog.md
D:\ycblog\content\post\buildhugoblog.md created

安装主题

$ cd themes/

$ git clone https://github.com/Vimux/Mainroad.git

回到站点根目录

$ cd ..

4 本地调试

$ hugo server -t Mainroad --buildDrafts

其中 –theme 选项可以指定主题，–buildDrafts 包括标记为草稿 然后在浏览器打开： http://localhost:1313

5 部署到Github

$ hugo -t Mainroad --baseUrl="https://ycsayit.github.io/" --buildDrafts

根目录下生成public文件夹

执行git 命令，将public目录添加远程仓库

浏览器打开：Git Pages
